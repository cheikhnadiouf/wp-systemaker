== Description ==

This simple plugin is a boilerplate, a starter kit to add custom codes in wordpress, an agile alternative to function.php in theme

== Installation ==


1. Upload the directory `wp-systemaker` to the `/wp-content/plugins/` directory or upload the zip file using the 'Uploads' feature in the Plugins dashboard.
2. Activate the plugin through the 'Plugins' menu in WordPress
